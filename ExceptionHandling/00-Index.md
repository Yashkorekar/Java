# Java Exception Handling Track

This folder contains runnable Q&A-style demos for Java exception handling, from first principles to interview-heavy edge cases.

## Suggested order
1. `TryCatchFinallyQnA.java`
2. `MultipleCatchBlocksQnA.java`
3. `CheckedVsUncheckedQnA.java`
4. `ThrowVsThrowsQnA.java`
5. `CustomExceptionsQnA.java`
6. `TryWithResourcesAndSuppressedExceptionsQnA.java`
7. `ExceptionPropagationAndBestPracticesQnA.java`

## What this folder covers well
- try, catch, finally control flow
- checked vs unchecked exceptions
- `throw` vs `throws`
- custom exceptions and cause chains
- multi-catch and catch ordering rules
- try-with-resources and suppressed exceptions
- propagation, translation, and handling best practices

## How to use these notes
- Run the demos and trace the control flow line by line.
- For each file, explain which exception reaches the caller and why.
- Learn the difference between handling, propagating, wrapping, and swallowing exceptions.
