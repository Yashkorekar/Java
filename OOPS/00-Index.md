# Java OOPS Track

This folder is a runnable OOP interview-prep track, starting from classes and objects and moving toward common design patterns and modern Java object-model features.

## Suggested order
1. `ClassesAndObjectsDemo.java`
2. `ConstructorValidationDemo.java`
3. `ConstructorOverloadingAndChainingQnA.java`
4. `ThisKeywordUsageQnA.java`
5. `GettersSettersQnA.java`
6. `MethodOverloadingVsOverridingQnA.java`
7. `OopsQnA.java`
8. `OopsInterviewTricksQnA.java`
9. `InterfacesAbstractClassesAndCompositionQnA.java`
10. `BuilderSingletonFactoryQnA.java`
11. `EnumsRecordsAndSealedClassesQnA.java`

## What this folder covers well
- classes, objects, state, behavior, references, aliasing
- constructors, chaining, validation, `this`, `super`
- encapsulation, getters/setters, defensive copying
- overloading, overriding, polymorphism, equality, object contract
- interfaces vs abstract classes, default methods, composition vs inheritance
- builder, singleton, and factory patterns
- enums, records, sealed classes, and modern Java OOP features

## How to use these notes
- Run the files and predict output before reading comments.
- For every topic, explain both the rule and the interview trap.
- If two concepts look similar, be able to contrast them precisely: overloading vs overriding, interface vs abstract class, composition vs inheritance.
