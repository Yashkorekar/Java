## Key versions (2026 reality)
- Spring Boot 3.x uses **Jakarta** namespaces (`jakarta.*`), not `javax.*`.
- Boot 3 baseline is **Java 17+**.

## How to use these notes
- Skim the “Tricky questions” sections first.
- Practice explaining **why** (not just what).
- For each topic, try to reproduce one mini-scenario in a small Boot app when you’re ready.
