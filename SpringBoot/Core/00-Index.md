- What happens during `SpringApplication.run()`.
- How component scanning, bean creation, scopes, proxies, and lifecycle callbacks work.
- How Boot binds configuration and decides which auto-configurations to apply.
- How Spring MVC handles requests, validation, serialization, and exceptions.
- How JPA sessions, transactions, lazy loading, and repositories behave.
